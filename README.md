## Sobre o projeto

Este repositório contém um jogo educativo simples construído com **HTML**, **CSS** e **JavaScript puro** (sem frameworks). O objetivo é ensinar conceitos básicos de teoria dos jogos, como equilíbrio de Nash, através de um cenário de competição de preços entre duas empresas.

- Usuário escolhe entre baixar, manter ou aumentar o preço.
- IA toma uma decisão baseada em histórico de jogadas.
- Resultados são exibidos a cada rodada com lucros e número de clientes.
- Ao final, uma explicação sobre o dilema do prisioneiro e equilíbrio de Nash é apresentada.

### Tecnologias

- Vite para bundling e deploy
- TailwindCSS (opcional) para estilos

### Como jogar

1. Acesse a versão online: https://jogo-mercado-competitivo-teoria-dos.vercel.app/
